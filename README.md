# Claude Actions Demo

A minimal Express app used to test Claude Code GitHub Actions.

## Setup

```bash
npm install
```

## Run

```bash
npm start        # starts server on http://localhost:3000
```

## Test

```bash
npm test
```

## Routes

| Method | Path      | Description        |
|--------|-----------|--------------------|
| GET    | /health   | Health check       |
| GET    | /items    | List items         |
| POST   | /signup   | Create a user      |
